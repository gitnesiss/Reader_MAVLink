# MAVLink Reader

## О проекте

Этот проект создан для решения одной из задач, на которой я учился работать с Wi-Fi соединением. По протоколу MAVLink должны передаваться данные по углам от полётного контроллера. При этом нужно было настраивать контроллер, чтобы обеспечить постоянство и определённую частоту передачи данных.
